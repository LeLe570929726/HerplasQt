# HerplasQt
A Qt GUI framwork with factory method.

### What is HerplasQt?
Qt is a powerful and easy framwork. It also can cross-platform. But it's UI edit part is not very friendly£¨I think :-(£©. So, this project is appear! In IDE part, you can make UI easily. Of cause, it's visible edit. Then, IDE can make an UI file. In code part, program can analyze the UI file and make the UI automatic in program. It's include Event system and Factory System and so on.

### What can HerplasQt do?
HerplasQt provide an Event System, a Factory System, a XML UI Processing System and so on. If you want to make a GUI program. First you can make a UI in HerplasQt IDE. Then the IDE can make a XML UI file automatic. Finally, you can use HerplasQt to read and processing the XML UI file. But it also only use a short code!

### What do you need?
HerplasQt is code in Qt5.4 and Visual Studio 2013. So you need Qt5.4, Visual Studio Add-in for Qt5 and Visual Studio 2013. You can download Qt 5.5 and Visual Studio Add-in for Qt5 from here: [Qt download](http://www.qt.io/download/). You can download Visual Studio 2013 from here: [Visual Studio Download](https://www.visualstudio.com/zh-cn/downloads/download-visual-studio-vs#).
